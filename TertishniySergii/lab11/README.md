## Мета роботи 
Познайомитися з групою твірних шаблонів проєктування
## Хід роботи
1. Створив [програму](lab11.py), в якій реалізував шаблон проєктування Factory Method.
Цей шаблон дозволяє створювати об’єкти певного типу через спеціальний метод-фабрику, не прив’язуючись безпосередньо до конкретних класів;
2. У програмі створено базовий клас Animal, від якого успадковуються два конкретні класи — Cat та Dog, що реалізують власний метод sound().
Далі створено клас AnimalFactory, який містить фабричний метод create_animal(). Він приймає назву тварини у вигляді рядка і повертає відповідний об’єкт. Завдяки цьому клієнтський код може працювати з тваринами, не знаючи, який саме клас створюється;
3. Приклад коду з реалізацією:
```python
class Animal:
    def sound(self):
        pass

class Cat(Animal):
    def sound(self):
        return "Meow"


class Dog(Animal):
    def sound(self):
        return "Woof"

#Factory Method
class AnimalFactory:
    def create_animal(self, animal_type):
        if animal_type == "cat":
            return Cat()
        elif animal_type == "dog":
            return Dog()
        else:
            return None

factory = AnimalFactory()

animal1 = factory.create_animal("cat")
animal2 = factory.create_animal("dog")
print(animal1.sound())  
print(animal2.sound())  
```
